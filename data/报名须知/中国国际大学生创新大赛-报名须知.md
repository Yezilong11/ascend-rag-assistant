# 中国国际大学生创新大赛（2025）报名须知

## 大赛基本信息

- **大赛主题**：我敢闯，我会创
- **报名官网**：https://cy.ncss.cn/
- **主办单位**：教育部、河南省人民政府
- **承办单位**：教育部学生服务与素质发展中心、国家大学生就业服务平台、各省高校毕业生就业网

## 报名时间安排

| 阶段 | 时间 |
|------|------|
| 参赛报名 | 4月-7月 |
| 高校初赛 | 4月-8月 |
| 省市复赛 | 4月-8月 |
| 总决赛 | 9月 |

## 参赛类别

### 高教主赛道
新工科类、新医科类、新农科类、新文科类、人工智能+、低空经济、生物技术、量子科技、新能源、新材料

### 青年红色筑梦之旅赛道
现代农业、制造业、信息技术服务、文化创意服务、社会服务

### 职教赛道
创新类、商业类、工匠类

## 报名方式

1. 访问全国大学生创业服务网：https://cy.ncss.cn/
2. 使用学信网账号登录
3. 点击"报名参赛"创建项目
4. 填写项目信息并报名参赛

## 报名注意事项

1. **账号要求**：需使用学信网账号登录报名
2. **身份选择**：选择身份后无法转换，请谨慎选择
3. **团队成员**：需确保团队成员在大学生创业服务网完成注册核验
4. **指导教师**：不要求必须有指导教师，可根据项目需要添加
5. **学院简称**：报名时请使用学院简称以便统计
