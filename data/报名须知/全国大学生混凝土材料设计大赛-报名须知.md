# 全国大学生混凝土材料设计大赛-报名须知

## 一、大赛基本信息
1. **主办单位**：中国混凝土与水泥制品协会(CCPA)教育与人力资源工作委员会、全国高等学校建筑材料学科研究会
2. **指导单位**：教育部高校材料类教学指导委员会（无机非金属材料专业教指委）
3. **举办周期**：每两年举办一次
4. **2026年第八届大赛信息**：
   - 时间：2026年7月17日-19日
   - 地点：甘肃兰州（西北民族大学承办）

## 二、参赛对象
全国各高校无机非金属材料工程、材料科学与工程、土木工程等相关专业的全日制本科生（含应届毕业生）。

## 三、报名方式
1. 参赛队伍以学校为单位报名，不接受个人单独报名
2. 每支参赛队伍由3-5名学生组成，需配备1-2名指导教师
3. 报名流程：
   - 关注学校教务处或相关学院发布的校内选拔赛通知
   - 在校内选拔赛中获得优异成绩的队伍将代表学校参加全国大赛
   - 按照大赛官方通知要求提交正式报名材料
4. 报名截止时间：各高校校内报名截止时间不同，通常在每年3-5月份，具体以学校通知为准（2026年哈尔滨工业大学校内报名截止时间为3月16日17:00）

## 四、比赛内容
大赛通常包括以下环节：
1. 理论知识考试（占比约40%）
2. 混凝土配合比设计
3. 现场实验操作
4. 作品展示与答辩

## 五、注意事项
1. 参赛队员必须为符合条件的在校本科生，不得冒名顶替
2. 配合比设计需符合大赛给定的主题和技术要求
3. 参赛队伍需遵守大赛各项规则，服从组委会安排
4. 具体报名要求和材料提交方式以当年发布的官方参赛通知为准
