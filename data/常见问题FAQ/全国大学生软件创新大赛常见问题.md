# 全国大学生软件创新大赛常见问题FAQ
## 一、报名相关问题
### Q1：参赛团队的人数要求是多少？
A：每支参赛队伍由3-5名在读学生组成，可以配备1-2名指导教师。

### Q2：不同学校的学生可以组成跨校团队参赛吗？
A：通常情况下不允许跨校组队，所有队员需来自同一所高校，具体以当年赛事规程为准。

### Q3：研究生和本科生可以混合组队吗？
A：可以，没有学历限制，只要是全日制在读学生均可组队。

### Q4：一个人可以参加多个团队吗？
A：不可以，每位学生只能加入一支参赛队伍。

## 二、作品相关问题
### Q1：参赛作品有什么要求？
A：1. 必须为团队原创，拥有完整知识产权，不得抄袭、盗用他人作品
2. 符合当年大赛主题和对应赛道的要求
3. 具有创新性、技术性和实用价值
4. 需提交完整的技术文档、演示视频、可运行程序等材料

### Q2：已经参加过其他比赛的作品可以参赛吗？
A：不可以。参赛作品必须是未在其他国家级竞赛中获奖的原创作品，具体要求以当年规定为准。

### Q3：作品提交后还可以修改吗？
A：在报名截止前通常可以修改，报名截止后无法再修改或更新作品，请提前做好准备。

## 三、比赛相关问题
### Q1：比赛需要提交哪些材料？
A：通常需要提交：
- 作品报告书（包括需求分析、设计方案、技术实现、创新点、应用前景等）
- 可运行的软件程序或演示系统
- 作品演示视频（5-10分钟）
- 其他补充材料（如专利、软著、相关获奖证明等）

### Q2：答辩环节有什么注意事项？
A：1. 提前准备好PPT和演示文稿，控制好时间（通常答辩10-15分钟，问答5分钟）
2. 突出作品的创新点和技术亮点
3. 提前进行多次模拟答辩，准备好评委可能问到的问题
4. 团队分工明确，答辩时互相配合

### Q3：比赛的评审标准是什么？
A：主要从以下几个方面评审：
1. 创新性：作品的创意和技术创新程度
2. 技术性：技术实现难度、技术先进性、系统稳定性
3. 实用性：应用价值、市场前景、可推广性
4. 完整性：文档完整性、演示效果、团队答辩表现

## 四、其他常见问题
### Q1：获奖证书有什么用？
A：本大赛是国家级A类竞赛，获奖证书在高校保研、综合测评、奖学金评定、就业等方面都有很高的认可度。

### Q2：指导老师有什么要求？
A：指导老师需为参赛队员所在高校的正式教师，每位老师可以指导多支队伍。

### Q3：参赛需要缴纳费用吗？
A：通常不需要缴纳报名费，参赛团队自行承担作品开发和差旅等相关费用。

### Q4：如何获取更多比赛相关信息？
A：可以关注示范性软件学院联盟官网、承办高校官网或大赛官方公众号，及时获取最新通知和赛事动态。

### Q5：软件安全攻防赛和软件作品赛有什么区别？
A：- 软件作品赛：侧重软件产品的创新设计与开发，需要提交完整的软件作品
- 软件安全攻防赛：侧重网络安全技术实战，考验选手的漏洞挖掘、攻防对抗能力，通常为现场竞技形式
