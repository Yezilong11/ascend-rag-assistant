# 全国大学生结构设计竞赛 常见问题

## 一、参赛资格类

### Q1: 全国大学生结构设计竞赛的参赛对象是什么？
**A:** 参赛对象为同一所高校在籍的全日制本科生、专科生。每支参赛队由3名学生组成，可指定1-2名指导教师。

### Q2: 研究生可以参加吗？
**A:** 本竞赛主要面向本科生和专科生。研究生参赛的具体要求请参考当年大赛通知。

### Q3: 可以跨校组队吗？
**A:** 不可以。参赛队的3名学生必须属于同一所高校，指导教师必须是参赛队所属高校的在职教师。

### Q4: 每所学校有多少参赛名额？
**A:** 
- 参加全国总决赛：每所高校推荐1个参赛队
- 承办全国总决赛的高校可推荐2个参赛队
- 各省分区赛可根据所属高校的数量规模，自行规定各高校的参赛队数

### Q5: 非土木工程专业可以参赛吗？
**A:** 可以的。虽然竞赛涉及结构力学等专业内容，但欢迎各专业对结构设计感兴趣的同学参加。每年都有跨专业团队获奖的案例。

## 二、报名流程类

### Q6: 如何报名参加全国大学生结构设计竞赛？
**A:** 报名流程一般如下：
1. 关注大赛官网或各省分区赛通知
2. 通过所在学校教务处或创新创业学院报名
3. 参加校内选拔赛
4. 通过校赛后参加省分区赛
5. 省赛获奖后推荐参加全国总决赛

### Q7: 报名需要准备什么材料？
**A:** 一般需要准备：
- 参赛报名表
- 团队成员信息
- 指导教师信息
- 学生证复印件

具体材料清单请以当年大赛通知为准。

### Q8: 学校没有组织校赛怎么办？
**A:** 建议联系所在学校的教务处、土木工程学院或创新创业学院咨询。如果学校没有组织校赛，可能需要通过其他渠道报名或前往其他高校参加分区赛。

### Q9: 报名截止时间是什么时候？
**A:** 一般每年的3-4月为校赛报名期，省分区赛报名在4-6月，具体时间请关注大赛官网或学校通知。

## 三、比赛内容类

### Q10: 竞赛题目一般是什么类型？
**A:** 竞赛题目通常结合实际工程背景，涵盖：
- 桥梁结构设计
- 高耸结构设计
- 空间结构设计
- 2025年题目为《乌篷船模型结构设计与制作》

### Q11: 比赛要求设计什么类型的结构？
**A:** 题目会要求针对水平荷载、竖向荷载及水平振动复杂工况进行受力分析、结构设计、模型制作及加载试验。

### Q12: 比赛需要完成哪些环节？
**A:** 竞赛过程包括：
1. 参赛报名
2. 提交方案
3. 开幕式
4. 现场制作模型
5. 陈述答辩
6. 加载测试
7. 专家点评
8. 闭幕式（颁奖仪式）

### Q13: 模型制作材料由谁提供？
**A:** 模型制作材料由竞赛组委会统一提供，参赛者不得自带规定外的材料，一经发现将取消参赛资格。

### Q14: 制作工具需要自己准备吗？
**A:** 是的，制作工具需要参赛者自行配备，包括：美工刀、剪刀、铅笔、直尺、三角尺、圆规、大头针等。

## 四、比赛赛程类

### Q15: 大赛每年举办几届？
**A:** 全国大学生结构设计竞赛每年举办一届。

### Q16: 比赛分为哪些阶段？
**A:** 比赛分为两个阶段：
1. **省（市、自治区）分区赛**（4月至7月上旬）
2. **全国总决赛**（一般安排在10月中下旬）

### Q17: 省分区赛和国赛有什么区别？
**A:** 
- 省分区赛：由各省教育行政部门或委托相关学会主办，高校轮流承办
- 全国总决赛：主要参赛高校包括发起高校、承办高校、各省推荐高校等

### Q18: 第十八届（2025年）总决赛在哪里举行？
**A:** 第十八届全国大学生结构设计总决赛定于2025年10月15日至19日在绍兴文理学院风则江校区举行。

## 五、奖项评选类

### Q19: 奖项是如何设置的？
**A:** 
- **省分区赛**：特等奖、一等奖、二等奖、三等奖
- **全国总决赛**：特等奖、一等奖、二等奖、三等奖、单项奖、优秀组织奖、突出贡献奖

### Q20: 获奖比例是多少？
**A:** 具体获奖比例由当年大赛组委会确定，一般而言：
- 一等奖约为入围决赛队伍的20%
- 二等奖约为35%
- 三等奖约为45%

### Q21: 获奖有什么奖励？
**A:** 
- 获得相应级别的获奖证书和奖杯
- 参赛经历有助于保研、出国、评选奖学金
- 成绩优秀者可代表学校参加更高级别比赛

### Q22: 如何知道是否入围下一阶段比赛？
**A:** 大赛官网会公布入围下一阶段的参赛队名单，同时也会通过各省份分区赛通知参赛队。

## 六、注意事项类

### Q23: 比赛时有什么特别需要注意的？
**A:** 
1. 必须使用竞赛组委会提供的材料
2. 需在规定时间内完成模型制作
3. 加载测试环节需注意安全
4. 答辩环节需准备充分

### Q24: 可以提前准备模型吗？
**A:** 全国总决赛一般采用现场制作的方式，参赛队需要在比赛现场使用统一材料进行模型制作。具体赛制请参考当年大赛通知。

### Q25: 指导教师有什么要求？
**A:** 指导教师必须是参赛队所属高校在职教师，每队1-2名。指导教师有责任保证参赛成员身份的真实性。

### Q26: 参赛需要交费吗？
**A:** 大赛本身不收取报名费用。但参赛期间的交通、食宿等费用需自理，具体费用政策请关注当年大赛通知。

## 七、备赛建议类

### Q27: 没有基础可以参赛吗？
**A:** 可以的。竞赛欢迎零基础、零经验的同学参加。意志坚定、乐于创新和有一颗不畏挑战的心才是获得良好成绩的关键。很多获奖选手都是通过赛前培训逐步掌握的。

### Q28: 如何准备结构设计大赛？
**A:** 建议：
1. 学习结构力学、材料力学基础知识
2. 参加学校组织的赛前培训
3. 学习往届优秀作品和赛题
4. 练习模型制作技能
5. 培养团队协作能力

### Q29: 往届大赛的题目在哪里查看？
**A:** 可以通过以下途径：
1. 大赛官网的历届回顾
2. 各省分区赛网站
3. 网络搜索往届赛题
4. 咨询有经验的学长学姐

### Q30: 参赛需要什么样的团队配置？
**A:** 建议团队配置：
- 至少一人具备较强的理论分析与建模能力
- 至少一人具备较强的动手制作能力
- 一人负责协调组织和答辩展示
- 鼓励跨学科、跨年级组队

## 九、赛制相关类

### Q34: 省分区赛和全国总决赛的赛制一样吗？
**A:** 
- 全国总决赛：一般采用提前公布题目，在同一时间、地点，使用统一材料、工具制作和统一设备测试
- 省分区赛：可根据实际情况，参照全国总决赛方式或采取其他竞赛方式

### Q35: 比赛评宙标准是什么？
**A:** 评审包括：
1. 方案设计与理论分析
2. 模型制作效果
3. 结构创新点
4. 现场陈述与答辩
5. 模型加载测试

## 十、其他问题

### Q36: 这个竞赛的认可度如何？
**A:** 全国大学生结构设计竞赛是：
- 教育部、财政部首次联合批准发文的全国性9大学科竞赛资助项目之一
- 第一批被中国高等教育学会纳入“全国普通高校大学生竞赛排行榜”统计名单的学科竞赛
- 土木工程相关学科培养大学生创新精神、团队意识和实践能力的最高水平学科竞赛

### Q37: 获奖对保研有帮助吗？
**A:** 是的，该竞赛获奖在保研中通常有较高认可度。具体加分政策请参考所在学校的相关规定。
